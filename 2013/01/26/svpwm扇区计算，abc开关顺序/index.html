SVPWM扇区计算，abc开关顺序

扇区 Ⅰ Ⅱ Ⅲ Ⅳ Ⅴ Ⅵ
扇区 $U0,U_{60}$ $U{60},U{120}$ $U{120},U{180}$ $U{180},U{240}$ $U{240},U{300}$ $U{300},U{360}$
N 3 1 5 4 6 2
$t_1$ -Z Z X -X -Y Y
$t_2$ X Y -Y Z -Z -X
$T_a$ a b c c b a
$T_b$ b a a b c c
$T_c$ c c b a a b

其中:
$$X=Uβ$$
$$Y=\frac{1}{2}(\sqrt{3}Uα + Uβ)$$
$$Z=\frac{1}{2}(-\sqrt{3}Uα + Uβ)$$
令
$$V{ref1}=X=Uβ$$
$$V{ref2}=-Z=\frac{1}{2}(\sqrt{3}Uα - Uβ)$$
$$V{ref3}=-Y=-\frac{1}{2}(\sqrt{3}Uα + Uβ)$$
求N值
$$V{ref1}>0，则A=1，否则A=0$$
$$V{ref2}>0，则B=1，否则B=0$$
$$V{ref3}>0，则C=1，否则C=0$$
N=4C+2B+A

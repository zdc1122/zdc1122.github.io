闲庭信步，领略江南风情——苏州


虎丘（东方的比萨斜塔）

七里山塘附近

与杭州、上海相比，苏州更具江南风味吧，上海是个国际化都市，杭州是介于苏州和上海之间的那种。苏州有更多的内河，所以在苏州能够更多感受到江南那种小桥流水的情调。记得很小的时候就上过课文讲的是苏州园林，苏州的却很多园林，但是如果一次全部都看完的话，会觉得没啥意思，可以一次看一个。本来还想去寒山寺的，结果那里在维修，留着今后去吧

去苏州游玩，可以多走走，沿着那些小街小巷，沿着河边，这个是区别的杭州的一点吧，这样才能感受到江南的风情，旅游是个慢的过程，如果旅游中还赶得急，往往会遗漏掉很多美好的东西

来过，便不曾离开——乌镇

很早就想去乌镇，因为一直听说乌镇，确从来没有去过，因为一个人不怎么喜欢出去跑，所以在学校几年也都没有去过什么地方，基本上都是呆在学校里的。

旅游的时候人最不喜欢的是人多，尤其去游览这些古镇的时候，这次选择的时间蛮好的，天气不热，人比较少。但是工作之后，估计这样的时间就不是很多了，除了假期，平时应该就很少有时间吧

江南的古镇中，乌镇是个经过商业开发过的一座古镇，添加了更多的现代化元素。

乌镇分为东栅和西栅，东栅是未开发的，也蛮小的，一小会就可以游览完，基本上都是民宅。游古镇需要的是一种心情吧，如果只是为旅游而旅游，就是一种走马观花式的，因为只有有了那种心情，才能去欣赏旅游中的景色，感受那里的美之所在。

Win7建立wifi功能

想好好利用下手机的wifi功能，手机连接电脑上网吗？

手机wifi是不是搜索不到笔记本建立的临时网络？

有两台电脑都想上网，只有一个宽带，没有路由器，怎么办？

现在就告诉大家一个简单有效的方法，在笔记本上建立一个wifi热点，供wifi手机或者其他笔记本链接

综述：其实就是三段代码:（括号内部分）

1.（netsh wlan set hostednetwork mode=allow ）

2.（netsh wlan set hostednetwork ssid=mywifi key=12345678 ）3.(netsh wlan start hostednetwork)

解释下，很简单，第一个就是设置网络承载模式为允许，第一个设置您建立的wifi用户名及密码，ssid=后面就是用户名，key=后面是密码，自己设定，第三个就开启了。
